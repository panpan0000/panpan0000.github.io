Meet RackHD Features

discover & catalog

Discovers the compute, network, and storage resources and catalogs their attributes and capabilities.

telemetry and genealogy

Telemetry data includes genealogical details.

PROVISIONING

Provisions a node to support the intended application workflow.

CONFIGURATION

Configures the hardware per application requirements: from the BIOS configuration of compute to the port configurations of network switches.

DEVICE MANAGEMENT

Powers devices on and off. Manages the firmware, power, OS installation, and base configuration of the resources.

FIRMWARE MANAGEMENT

Manages all infrastructure firmware versioning, including configuration and firmware update.

LOGGING

Log information can be retrieved for particular elements or collated into a single timeline for multiple elements within the management neighborhood.

ENVIRONMENTAL MONITORING

Aggregates environmental data from hardware resources.

DATA ALALYTIC

(Vision) Data generated by environmental and fault monitoring can be provided to analytic tools for analysis, particularly around predictive failure

FAULT DETECTION

(Vision) Monitors compute and storage devices for both hard and soft faults. Performs suitable responses based on pre-defined policies.
